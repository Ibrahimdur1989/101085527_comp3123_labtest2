# 101085527_comp3123_labtest2 - Weather App
COMP3123 - Full Stack Development I

Created by: **Ebrahim Al-Serri (101085527)**


Simple React weather application for COMP3123 Lab Test 2


## ✨ How to Run:- 
1. Install packages:
npm install

2. Create a .env file in the project root:
REACT_APP_WEATHER_API_KEY=your_api_key_here

3. Start the app:
npm start


## ✨ API Used
OpenWeatherMap - Current Weather API
